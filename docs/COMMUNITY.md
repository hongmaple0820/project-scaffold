# 社区与分发

语言：中文 | [English](COMMUNITY.en.md)

project-scaffold 是工程化工作流实践脚手架，用来把 SCALE Engine 的治理基线复制到新项目。社区与分发入口统一维护在这里，供脚手架、派生项目和推广材料复用。

## 参与方式

- Star / Watch [scale-engine](https://github.com/hongmaple0820/scale-engine)，关注 CLI、治理模板和工作流能力更新。
- 通过 GitHub Issues 反馈模板、Agent 工作流、质量门禁或项目适配问题。
- 通过 PR 贡献跨语言模板、验证脚本、资源治理规则或示例项目。
- 国内用户可通过 [Gitee 镜像](https://gitee.com/hongmaple/scale-engine) 查看同步内容。
