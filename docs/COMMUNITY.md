# 社区与赞助

project-scaffold 是 SCALE Engine 工程治理实践的项目脚手架沉淀，目标是让新项目快速获得可执行工作流、质量门禁、任务产物和工程规范。

## 社区入口

- GitHub: [hongmaple0820/scale-engine](https://github.com/hongmaple0820/scale-engine)
- Gitee: [hongmaple/scale-engine](https://gitee.com/hongmaple/scale-engine)
- npm: [@hongmaple0820/scale-engine](https://www.npmjs.com/package/@hongmaple0820/scale-engine)

## 支持方式

- Star 项目，帮助更多开发者发现这套工作流。
- 提交 issue，反馈脚手架、Agent 工作流、质量门禁或模板问题。
- 贡献 PR，补充跨语言项目治理模板。
- 通过 [GitHub Sponsors](https://github.com/sponsors/hongmaple0820) 打赏支持持续维护。
